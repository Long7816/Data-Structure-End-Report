# 智行北科 (OmniNTUT)：結合下雨/高溫/空污/YouBike 的校園動態路徑規劃系統

本專案是「智行北科 (OmniNTUT)：結合下雨/高溫/空污/YouBike 的校園動態路徑規劃系統」的前端展示 Demo 靜態網站。

## 🌟 專案特色

*   **校園與周邊導航路網**：以圖形結構（Graph）建模捷運站、宿舍、各教學大樓與教室，並結合 5 個實體 YouBike 站點。
*   **YouBike 狀態動態監控**：當可用車輛或空位數低於 3 時觸發黃/紅燈臨界警告，演算法會自動繞道或導引至鄰近替代站點。
*   **動態環境變數模擬**：可點擊切換烈日高溫（38°C 避曝曬）、突然暴雨（路滑避雨）與空氣污染（避主要幹道 AQI 廢氣）等情境。
*   **幸福通勤最佳路徑**：支援 6 種通勤與避險模式切換，透過權重處罰（Penalty）機制即時計劃最佳路徑。
*   **視覺化對比與解說**：地圖同時呈現「原始最短路徑（紅虛線）」與「幸福優化路徑（綠發光線）」，並附帶資料結構與運算複雜度解說面板。

## 🛠️ 技術棧

*   HTML5 / CSS3 (Vanilla CSS) / JavaScript (ES6)
*   無後端、無資料庫、免安裝，雙擊開啟 `index.html` 即可完整運行。
