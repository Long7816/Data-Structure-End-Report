# 智行北科 (OmniNTUT) - 產品需求文件 (PRD)

## 1. 專案背景與目標
本專案為「智行北科 (OmniNTUT)：結合下雨/高溫/YouBike 的校園動態路徑規劃系統」的前端展示網頁。旨在結合真實校園地理資訊（北科大校園），透過圖形結構與戴克斯特拉（Dijkstra）演算法，為校園師生、來訪貴賓規劃在不同天候情境（突然暴雨、烈日高溫、YouBike車位供需失衡）下的「最佳幸福通勤路徑」。

## 2. 核心技術架構 (Tech Stack)
* **前端技術**：純 HTML5, Vanilla CSS3, JavaScript (ES6+)。
* **架構限制**：無後端伺服器、無資料庫、無登入系統、不使用任何 NPM 套件與現代框架 (React/Vue/Next/Vite 等)。
* **地圖渲染**：使用 SVG 繪製節點與路網邊線，支援流光動畫與動態多邊形天候遮罩。
* **部屬與執行**：使用者可直接點擊 `index.html` 離線執行。

## 3. 功能需求

### 3.1 校園路網地圖 (Campus Graph)
* **節點 (Nodes)**：
  * 必須採用真實經緯度投影至 $1000 \times 800$ 的 SVG 坐標系。
  * 包含：捷運站、大樓門口、宿舍、YouBike 站點、路口。
  * 支援滑鼠點擊節點快速切換為導航終點，增強人機互動性。
* **邊 (Edges/Paths)**：
  * 包含物理距離、是否具備樹蔭、是否具備雨遮、坡度等級（平坦/微坡/陡坡與樓梯）、路面材質（柏油/大理石/紅磚）。

### 3.2 YouBike 監控與臨界值預警
* **動態監控**：模擬臺北市開放資料的車輛與車位狀態。
* **臨界預警**：當站點可用車輛或空位 $< 3$ 時，判定為高風險站點，地圖上呈現紅色呼吸燈警告。
* **自動繞道**：在「騎車轉乘」模式下，若起點站點無車可借，或目的地站點滿位無法還，演算法會自動尋找鄰近備用站點進行導航。

### 3.3 動態環境影響區域 (Dynamic Hazard Zones)
* **烈日高溫區**：在地圖渲染橘黃色半透明多邊形，遮蓋無遮蔭路段。
* **突然暴雨區**：在地圖渲染藍色半透明多邊形，遮蓋無雨遮露天路段。

### 3.4 幸福通勤最佳導航路線
* 系統提供多種偏好模式，透過動態調高（Penalty）特定邊的權重計算出不同路徑，並在地圖上同時呈現「物理最快路徑」與「優化幸福路徑」以利對比：
  1. **最快路線**：物理最短距離。
  2. **最涼避曝曬**：高溫下避開無遮蔭路段（罰值增加 250%）。
  3. **少淋雨走廊**：雨天避開無雨遮露天路段（罰值增加 300%）與易滑大理石面。
  4. **無障礙避陡坡**：避開坡度為 2~3（樓梯）的路段（樓梯權重放大 12 倍）。
  5. **騎車轉乘**：步行與騎單車複合路線。

### 3.5 演算法與資料結構可視化說明
* **Hash Map 應用**：以 $O(1)$ 時間複雜度更新、查詢 YouBike 狀態與各邊之動態 Penalty 加權乘數。
* **Dijkstra 演算法**：即時分析邊的動態加權，渲染優化路徑，並在側邊監控面板詳細說明 Penalty 加權原理。

## 4. 驗收標準 (Acceptance Criteria)
* **無錯誤執行**：雙擊 `index.html` 開啟，無任何 Console 錯誤。
* **3 分鐘簡報展示路徑**：
  * 起點：「捷運忠孝新生站(4號出口)」
  * 終點：「科研大樓」
  * 可流暢演示切換「突然暴雨」+「少淋雨模式」繞道、以及「YouBike車位吃緊」下自動改至鄰近站點借還車的完整決策過程。
* **高質感 UI**：玻璃磨砂科技風，亮眼吸睛，適合期末報告。
