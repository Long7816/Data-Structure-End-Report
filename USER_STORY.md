User Story 1：機車通勤族與 YouBike 轉乘（避雨需求）
  •	作為一位 從外地騎車通勤到學校的學生， 
    想要在停好機車準備轉乘 YouBike 到教室前，查詢一條「少淋雨」的導航路線並確認沿途站點是否有車，  
    因為天氣突然開始下毛毛雨，我不想一早進教室上課就全身濕答答。  
    
User Story 2：炎夏宿舍出發的學生or住學校附近的學生（防曬/高溫需求）
  •	作為一位住在學校宿舍、下午要前往遠側教學大樓上課的學生，
    想要規劃一條走廊或樹蔭較多、體感「最涼」的步行路線，
    因為夏天正午的太陽太毒辣，我想避免在毫無遮蔽的柏油路上曝曬而中暑。
    
User Story 3：行動不便或推行李的校園成員（無障礙需求）
  •	作為一位剛好腳部受傷需要拄拐杖、或是需要推重物進校園的學生，
    想要在導航時直接切換到「無障礙」路線模式，避開所有樓梯、高陡坡或崎嶇不平的路面，
    因為確保沿途都有斜坡或電梯，能讓我安全、省力且幸福地抵達教室。
    
User Story 4：趕課或早八遲到邊緣人（最快路徑需求）
  •	作為一位 早上快要遲到、剛從校門口要衝向教室的學生，
    想要系統能利用 Dijkstra 或 A* 演算法幫我精準算出當下「最快」抵達教室的捷徑，
    因為教授這堂課會準時點名，差一分鐘我就會被記缺席，此時速度比任何天氣因素都更重要。
